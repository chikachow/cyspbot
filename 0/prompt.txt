## System and assets

cyspbot is a pair of public Cloudflare Workers. The token-exchange Worker verifies GitHub Actions OIDC assertions, applies checked-in authorization policy, and mints repository-scoped GitHub App installation tokens. The webhook Worker authenticates GitHub deliveries before acknowledging them. The central objective is that no remote caller can mint a token unless a signature-verified assertion from the configured issuer yields a consistent GitHub Actions principal and an explicit policy rule permits the exact repository and permissions.

## Trust boundaries

1. **Internet to Workers.** Paths, methods, headers, bodies, form fields, JWTs, claims, webhook payloads, and forwarding metadata are attacker-controlled. Parsing, size limits, and failure handling must remain fail-closed.
